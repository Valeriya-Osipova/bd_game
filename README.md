# Ведро для монашки

Минимальная браузерная игра в жанре "тяжёлой работы": монашка бегает между колодцем и бочкой, наполняет ведро у колодца, носит его к бочке и пытается заполнить её.

## Запуск

Проект работает как обычная статическая веб-страница. Самый простой способ запустить его локально:

```bash
python -m http.server 8000
```

После этого откройте в браузере:

```text
http://127.0.0.1:8000/
```

## Управление

- Стрелки влево/вправо или A / D — движение по линии
- Пробел — наполнять ведро у колодца
- E — вылить воду в бочку

## Сборка и хостинг

Проект можно запустить локально и собрать в статическую папку `dist`.

```bash
npm install
npm run dev
```

Для подготовки релиза:

```bash
npm run build
```

Готовая сборка появится в папке `dist/` и подходит для статического хостинга (GitHub Pages, Netlify, Vercel и т.п.).

## Docker

Если хочешь упаковать игру в контейнер, добавлен `Dockerfile`:

```bash
docker build -t vedro-dlya-monashki .
```

Запуск контейнера:

```bash
docker run --rm -p 8080:80 vedro-dlya-monashki
```

После этого игра будет доступна по адресу `http://localhost:8080`.

## GitHub Pages

1. Залей проект в репозиторий на GitHub.
2. Перейди в `Settings → Pages`.
3. Выбери ветку `main` и папку `/dist`.
4. Сохрани изменения — GitHub выдаст ссылку для игры.
